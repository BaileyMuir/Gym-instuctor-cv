summary

the content of these programs has been desighned to use freewights and can be altered for machines and free weight in some cases, each split has been seperated in a way in which the key muscles that work together are placed together in the split such as chest and tricep as a result of the movement to uterlise it is a pushing one that also engages the triceps.


if you wanted to make it easyer you could put the bicep exercises with the chest if it is to hard to start with but it wont uterlise the muscles and pushg them as much as if you followed the planned program. on that note the legs have few exercises as the calfs will be uterlised while running on a treadmill and during daily life so i tryed not to overwork it adding to the fact alot of exercises to target individual muscles require weight machines with that in mind the use of core muscles exercises was used as a way to finish this work out.
