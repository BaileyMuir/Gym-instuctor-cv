Summary

The content of these programs has been designed to use free weights and can be altered for machines and bodyweight in some cases, and each split has been separated in a way in which the key muscles that work together are placed together in the split, such as chest and tricep as a result of the movement to utilise it is a pushing motion that also engages the triceps.


If you wanted to make it easier, you could put the bicep exercises with the chest if it is too hard to start with, but it won't utilise the muscles and push them as much as if you followed the planned program. On that note, the legs have few exercises as the calves will be utilised while running on a treadmill and during daily life, so I tried not to overwork it and only added one which is really optional, adding to the fact a lot of exercises to target individual muscles require weight machines with that in mind the use of core muscles exercises was used as a way to finish this workout.
